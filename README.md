# Omkar U. Parmekar - Personal Portfolio

A modern, responsive personal portfolio website built to showcase my skills and experience as a **Data Analyst** and **Cyber Security Analyst**.

## Features
- **Glassmorphism Design:** Modern UI with semi-transparent cards and backdrop blurs.
- **Dark / Light Mode:** A fully functional theme toggle that saves user preferences to local storage.
- **Interactive UI:** Smooth hover effects, scroll-reveal animations, and a sticky navigation bar.
- **Dual Resumes:** Dedicated download buttons for both Data Analyst and Cyber Security Analyst roles.
- **Responsive Layout:** Optimized for mobile, tablet, and desktop viewing.

## Technologies Used
- HTML5
- CSS3 (Vanilla, with CSS Variables for theming)
- JavaScript (Vanilla)
- Boxicons (Icons)
- ScrollReveal.js (Animations)

## Setup and Usage
1. Clone or download the repository.
2. Open `index.html` in any modern web browser.
3. Use the toggle in the header to switch between Dark and Light mode.
4. Click the respective buttons in the Hero section to download either the Data Analyst or Cyber Security resume.
